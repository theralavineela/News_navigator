# News_navigator
This repository is my code for ET GEN AI HACKATHON 
🧭 News Navigator AI

##  Problem
Users consume fragmented news from multiple sources, leading to:
- Information overload
- Bias and lack of perspective
- Time-consuming analysis

##  Solution
News Navigator AI aggregates real-time news and uses AI to:
- Summarize key developments
- Highlight multiple perspectives
- Analyze economic/market impact
- Generate follow-up insights

## Features
- Real-time news fetching (NewsAPI)
- AI summarization (Groq LLM)
- Structured briefing output
- Fallback mechanism for reliability

##  Tech Stack
- FastAPI (Backend)
- Streamlit (Frontend)(version1)
- NewsAPI (Data)
- Groq API (AI)
- Anti gravity(Frontend app deployment)
