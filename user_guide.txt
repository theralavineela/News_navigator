USER GUIDE — News Navigator AI
 Overview

News Navigator AI helps users quickly understand any topic by converting real-time news into structured insights like summaries, key points, and market impact.

Who Can Use This?
Students & Researchers
Business Analysts
General Users

 How to Use

Step 1: Launch the Application
- Start backend server
- Run frontend (Streamlit)
- Open app in browser

Step 2: Enter a Topic

In the input box, type any topic such as:

“AI in healthcare”
“India economy”
“Tesla stock”

Step 3: Click Analyze

The system will:
- Fetch latest news
- Process it using AI

Step 4: View Output

You will get:

✅ Summary — Quick overview
✅ Key Points — Important highlights
✅ Perspectives — Different viewpoints
✅ Market Impact — Economic/business effects
✅ Follow-up Questions — For deeper thinking
